# GrabWP Tenancy v1.1.3

_Released: 2026-07-10 · Previous: v1.1.2_

## Highlights
- CLI tenant detection via `GRABWP_TENANCY_TENANT_ID` env var with auto HTTP_HOST/HTTPS resolution
- WP-CLI `$table_prefix` synchronization fix for eval-scope stale prefix
- CDN URL propagation in database export metadata

## Enhancements
- Tenant detection supports `GRABWP_TENANCY_TENANT_ID` env var for CLI boot without wp-config constants
- CLI tenant context auto-populates `HTTP_HOST` and `HTTPS` from tenant mappings
- WP-CLI `$table_prefix` file-scope global re-bind prevents stale prefix after eval
- CDN URL included in database export metadata when Pro storage is configured
- Tenant list table uses natural column widths for extension compatibility
- Responsive CSS for tenant list on mobile screens

## Fixes
- `UPLOADS` constant only defined when upload dir is inside ABSPATH, preventing path errors on custom layouts
- Server info cache resettable via `grabwp_tenancy_reset_server_info_cache()` after environment changes
